Employee management Web API

Provides CRUD operations to manage employees

The Web API service enables HTTP clients to add, delete, update and search employees.

Description of API:

  • Add employee
  • Delete employee
  • Update employee
  • Search employee

Employee

Combination of name and surname is unique, grade is in the range from 1 to 10 and salary is a positive integer or zero. Id is auto-generated when a new employee is added.
Search is not case-sensitive.

Examples

  • Searching for an employee:

    http://localhost:8080/employees/search?name=Amy
    http://localhost:8080/employees/search?name=amy
    http://localhost:8080/employees/search?grade=1
    http://localhost:8080/employees/search?grade=1&salary=0
    http://localhost:8080/employees/search?name=John&surname=Smith

Error messages

  • In the case of lack of data for a correctly determined data or incorrectly formulated enquiries, the service returns 404 Not Found message
